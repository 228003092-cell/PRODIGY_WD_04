SocialConnect
SocialConnect is a simple social media web page built with HTML, CSS, and JavaScript. Users can create posts with text or images, like and comment on posts, and follow suggested users.

Features:
  Create posts with text and images
  Like and comment on posts
  Follow/unfollow suggested users
  View trending hashtags
  Interactive and dynamic feed (posts appear instantly)
  User-friendly interface with a clean layout
  Left sidebar for navigation (Home, Explore, Alerts, Profile)
  Right sidebar for suggested follows and trending topics
  Customizable post input (username and text)
  Responsive layout using flexbox
